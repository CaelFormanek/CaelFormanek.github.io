My Projects

1. Mile High Hype

Mile High Hype is a web application that pulls sports news articles for various teams from external sources using a scraper. The backend is built with Django and provides a REST API for managing articles, including creating, retrieving, updating, and deleting news entries. The frontend is developed with React, which fetches the latest articles from the API and displays them in a clean, responsive layout. Each article is color-coded based on the team, with details such as the article title, publication date, and content displayed in well-structured containers. The application is designed to offer real-time updates and present sports news in an engaging and user-friendly format.

2. Project Two

Project Two is another web-based application that utilizes a different technology stack, focusing on real-time data processing and user engagement. It uses frameworks such as Angular for the frontend, Node.js for the backend, and MongoDB as the database. The app features user authentication, real-time chat, and personalized recommendations.

3. Project Three

Project Three is an innovative application aimed at streamlining business processes through automation. The app integrates various APIs to automate workflows, reduce manual intervention, and enhance productivity. The frontend is designed with Vue.js, and the backend utilizes a Python-based framework. The project includes a dashboard for tracking key metrics and a notification system for real-time updates.
